class nib.views.shapes.RoundedRectangle(corner_radius=10, fill=None, stroke_color=None, stroke_width=None, **kwargs)[source]

Bases: View

A rectangular shape with rounded corners.

RoundedRectangle draws a rectangle with configurable corner rounding. It is commonly used for card backgrounds, button shapes, and container styling. The corner radius can be set to any positive value, where larger values create more pronounced rounding.

When used as a clip_shape for other views, RoundedRectangle clips the content to the rounded rectangle bounds.

_type

The SwiftUI view type identifier (RoundedRectangle).

_corner_radius

The corner radius in points.

_stroke_color

The stroke color if set.

_stroke_width

The stroke width if set.

Parameters:

  • corner_radius (float) The radius of the rounded corners in points. Defaults to 10. Larger values create more rounded corners.

  • fill (Union[str, Color, None]) The fill color for the rectangle interior. Can be a Color instance, a named color string (e.g., red), or a hex color string (e.g., #FF5733).

  • stroke_color (Union[str, Color, None]) The stroke color for the rectangle border. Uses the same color format as fill.

  • stroke_width (Optional[float]) The width of the stroke in points. Defaults to 1 if stroke_color is set but stroke_width is not.

  • **kwargs (Any) Additional view modifiers passed to the base View class (e.g., width, height, padding, opacity).

Example

Simple rounded rectangle:

nib.RoundedRectangle(corner_radius=10, fill=nib.Color.BLUE)

Card with border:

nib.RoundedRectangle(
    corner_radius=16,
    fill=nib.Color.BLUE,
    stroke_color=nib.Color.WHITE,
    stroke_width=2,
    width=200,
    height=100,
)

As a clip shape for an image:

nib.Image(
    source="avatar.jpg",
    clip_shape=nib.RoundedRectangle(corner_radius=8),
    width=64,
    height=64,
)

See also

Rectangle: For sharp-cornered rectangles. Capsule: For fully rounded ends (pill shape).

class nib.views.shapes.Circle(fill=None, stroke_color=None, stroke_width=None, trim_from=None, trim_to=None, rotation=None, **kwargs)[source]

Bases: View

A circle centered in the frame of the view containing it.

Circle draws a circular shape that fills the smaller dimension of its frame (always maintaining a 1:1 aspect ratio). It can be used as standalone content, as a clip shape for images, or with trim properties to create progress rings and arc indicators.

The trim properties (trim_from and trim_to) allow drawing partial circles, useful for progress indicators. Values range from 0.0 to 1.0, where 0.0 is the start of the circle (3 oclock position by default) and 1.0 is a complete circle. Use rotation to change the starting position (e.g., -90 to start from the top).

_type

The SwiftUI view type identifier (Circle).

_trim_from

The starting point for trimmed circles (0.0-1.0).

_trim_to

The ending point for trimmed circles (0.0-1.0).

_rotation

The rotation angle in degrees.

Parameters:

  • fill (Union[str, Color, None]) The fill color for the circle interior. Can be a Color instance, a named color string (e.g., red), or a hex color string (e.g., #FF5733).

  • stroke_color (Union[str, Color, None]) The stroke color for the circle border. Uses the same color format as fill.

  • stroke_width (Optional[float]) The width of the stroke in points. Only applies if stroke_color is set.

  • trim_from (Optional[float]) The fractional starting point for the circle arc, from 0.0 to 1.0. Used with trim_to for progress rings.

  • trim_to (Optional[float]) The fractional ending point for the circle arc, from 0.0 to 1.0. Used with trim_from for progress rings.

  • rotation (Optional[float]) The rotation angle in degrees. Useful for positioning the start point of trimmed circles (e.g., -90 for top).

  • **kwargs (Any) Additional view modifiers passed to the base View class (e.g., width, height, padding, opacity).

Example

Simple filled circle:

nib.Circle(fill=nib.Color.BLUE)

Circle with stroke and dimensions:

nib.Circle(
    fill=nib.Color.RED,
    stroke_color=nib.Color.WHITE,
    stroke_width=2,
    width=50,
    height=50,
)

Progress ring (75% complete):

nib.Circle(
    stroke_color=nib.Color.BLUE,
    stroke_width=8,
    trim_from=0.0,
    trim_to=0.75,
    width=200,
    height=200,
)

Progress ring starting from top:

nib.Circle(
    stroke_color=nib.Color.GREEN,
    stroke_width=6,
    trim_from=0.0,
    trim_to=progress,  # 0.0 to 1.0
    rotation=-90,
    width=100,
    height=100,
)

See also

Ellipse: For oval shapes with different width and height.

Initialize a Circle shape.

Parameters:

  • fill (Union[str, Color, None]) The fill color for the circle interior. Accepts a Color instance, named color string, or hex string.

  • stroke_color (Union[str, Color, None]) The stroke color for the circle border.

  • stroke_width (Optional[float]) The width of the stroke in points.

  • trim_from (Optional[float]) The fractional starting point (0.0-1.0) for partial circle arcs. Defaults to None (full circle).

  • trim_to (Optional[float]) The fractional ending point (0.0-1.0) for partial circle arcs. Defaults to None (full circle).

  • rotation (Optional[float]) The rotation angle in degrees. Use -90 to start trimmed circles from the top instead of the right.

  • **kwargs (Any) Additional view modifiers (width, height, padding, opacity, etc.) passed to the base View class.

class nib.views.shapes.Rectangle(fill=None, stroke_color=None, stroke_width=None, **kwargs)[source]

Bases: View

A rectangular shape aligned inside the frame of the view containing it.

Rectangle draws a sharp-cornered rectangular shape that fills its frame. It can be used as standalone content, as a background for other views, or as a clip shape. Supports fill colors, strokes, and all standard view modifiers.

_type

The SwiftUI view type identifier (Rectangle).

Parameters:

  • fill (Union[str, Color, None]) The fill color for the rectangle interior. Can be a Color instance, a named color string (e.g., red), or a hex color string (e.g., #FF5733).

  • stroke_color (Union[str, Color, None]) The stroke color for the rectangle border. Uses the same color format as fill.

  • stroke_width (Optional[float]) The width of the stroke in points. Only applies if stroke_color is set.

  • **kwargs (Any) Additional view modifiers passed to the base View class (e.g., width, height, padding, opacity).

Example

Simple filled rectangle:

nib.Rectangle(fill=nib.Color.BLUE)

Rectangle with specific dimensions and stroke:

nib.Rectangle(
    fill=nib.Color.GRAY,
    stroke_color=nib.Color.BLACK,
    stroke_width=1,
    width=100,
    height=50,
)

Semi-transparent rectangle overlay:

nib.Rectangle(
    fill=nib.Color.BLACK.with_opacity(0.5),
    width=200,
    height=200,
)

See also

RoundedRectangle: For rectangles with rounded corners.

class nib.views.shapes.Ellipse(fill=None, stroke_color=None, stroke_width=None, **kwargs)[source]

Bases: View

An ellipse aligned inside the frame of the view containing it.

Ellipse draws an oval shape that stretches to fill its frame. Unlike Circle which maintains a 1:1 aspect ratio, Ellipse can have different width and height values, creating an oval shape.

When width equals height, an Ellipse appears identical to a Circle. For asymmetric shapes, the ellipse stretches along the larger dimension.

_type

The SwiftUI view type identifier (Ellipse).

Parameters:

class nib.views.shapes.Capsule(fill=None, stroke_color=None, stroke_width=None, **kwargs)[source]

Bases: View

A capsule shape aligned inside the frame of the view containing it.

Capsule draws a pill-shaped rectangle where the shorter dimension (width or height) determines the corner radius, creating fully rounded ends. This is different from RoundedRectangle which has a fixed corner radius regardless of dimensions.

For horizontal capsules (width > height), the ends are semicircles with radius equal to half the height. For vertical capsules (height > width), the ends are semicircles with radius equal to half the width.

_type

The SwiftUI view type identifier (Capsule).

Parameters:

  • fill (Union[str, Color, None]) The fill color for the capsule interior. Can be a Color instance, a named color string (e.g., red), or a hex color string (e.g., #FF5733).

  • stroke_color (Union[str, Color, None]) The stroke color for the capsule border. Uses the same color format as fill.

  • stroke_width (Optional[float]) The width of the stroke in points. Only applies if stroke_color is set.

  • **kwargs (Any) Additional view modifiers passed to the base View class (e.g., width, height, padding, opacity).

Example

Simple capsule:

nib.Capsule(fill=nib.Color.GREEN, width=100, height=40)

Capsule with border:

nib.Capsule(
    fill=nib.Color.BLUE,
    stroke_color=nib.Color.WHITE,
    stroke_width=2,
    width=150,
    height=50,
)

Vertical capsule:

    fill=nib.Color.ORANGE,
    width=30,
    height=100,

class nib.views.shapes.LinearGradient(colors=None, stops=None, start=(0.5, 0), end=(0.5, 1), **kwargs)[source]

Bases: View

A gradient that varies colors along a line from start to end points.

LinearGradient creates a smooth color transition along a straight line. The start and end points are specified in unit coordinates (0-1) where (0, 0) is top-left and (1, 1) is bottom-right.

Parameters:

  • colors (Optional[list[Union[str, Color]]]) List of colors for the gradient, evenly distributed. Use this OR stops, not both.

  • stops (Optional[list[tuple[float, Union[str, Color]]]]) List of (position, color) tuples for explicit control. Positions are 0.0-1.0. Use this OR colors, not both.

  • start (tuple[float, float]) Start point as (x, y) in unit coordinates. Default (0.5, 0) is top-center.

  • end (tuple[float, float]) End point as (x, y) in unit coordinates. Default (0.5, 1) is bottom-center.

  • **kwargs (Any) Additional view modifiers.

Example

Simple two-color gradient:

nib.LinearGradient(colors=["red", "blue"])

Diagonal gradient with explicit stops:

nib.LinearGradient(
    stops=[(0.0, "red"), (0.5, "yellow"), (1.0, "blue")],
    start=(0, 0),
    end=(1, 1),
)
