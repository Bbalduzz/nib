# Effects

Effect views apply visual transformations and rendering effects to content. These views leverage macOS system APIs for hardware-accelerated visual effects.

| View | Description |
|------|-------------|
| [VisualEffectBlur](blur.md) | Applies a macOS frosted-glass blur (NSVisualEffectView) with configurable material and blending mode. |
| [Canvas](canvas.md) | A Core Graphics drawing surface with gesture support for freeform 2D rendering. |
