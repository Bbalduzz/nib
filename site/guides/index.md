# Guides

Practical guides for building macOS menu bar applications with Nib. Each guide covers a specific topic with code examples you can copy into your project.

| Guide | Description |
|-------|-------------|
| [Building Layouts](layouts.md) | Stack, scroll, list, form, grid, and spacer containers |
| [Styling & Theming](styling.md) | Backgrounds, borders, shadows, opacity, corner radius, and clip shapes |
| [Colors & Gradients](colors-gradients.md) | Named colors, hex, RGB, RGBA, and linear/radial/angular gradients |
| [Typography & Fonts](typography.md) | System fonts, custom fonts, font weights, text styles, and attributed strings |
| [Animations & Transitions](animations.md) | Timing curves, springs, content transitions, and view transitions |
| [Context Menu](context-menu.md) | Right-click menu items, submenus, shortcuts, badges, and custom views |
| [Settings & Persistence](settings.md) | Settings class, UserDefaults, settings pages with tabs |
| [Notifications](notifications.md) | Push and scheduled notifications, sounds, actions, and text input |
| [System Services](services.md) | Battery, connectivity, screen, keychain, camera, launch at login, permissions |
| [File Dialogs](file-dialogs.md) | Open, save, and directory picker dialogs with native NSOpenPanel/NSSavePanel |
| [Canvas Drawing](canvas.md) | Core Graphics drawing surface with shapes, gradients, text, and gesture handling |
| [Charts](charts.md) | Swift Charts integration with line, bar, area, point, sector, and rule marks |
| [Navigation](navigation.md) | NavigationStack, NavigationLink, and DisclosureGroup for hierarchical navigation |
| [Drag & Drop](drag-drop.md) | File drag-and-drop handling on any container view |
| [Hotkeys & Clipboard](hotkeys-clipboard.md) | Global keyboard shortcuts and system clipboard access |
| [Custom Fonts](custom-fonts.md) | Loading custom TTF/OTF fonts from assets, paths, or URLs |
| [Assets](assets.md) | Asset directory, resolution, and bundled mode support |
